Teen Coding Workshop

Unlock the world of programming! This hands-on workshop is designed for teens eager to learn basic coding concepts and create their first simple project.

Date: 16 August 2025

Time: 3:00 PM - 5:00 PM

Location: Library Computer Lab (Level 3)

Instructor: Mr. Ben Lee (Software Engineer)

Ages: 13 - 17 years old

Requirements: No prior coding experience needed. Laptops provided.

In this workshop, participants will:

  • Learn fundamental programming concepts (variables, loops, conditionals)
  • Be introduced to a beginner-friendly coding language (e.g., Python or Scratch)
  • Build a small interactive program or game
  • Develop problem-solving and logical thinking skills
